    ## Getting Started

    To start using the materials:

    1. Clone the repository and change into it:
       ```
       git clone https://github.com/SdIshtiyaqAhmed/AIAPSUP-Task-Sets.git
       cd AIAPSUP-Task-Sets
       ```
    2. Open assignment or task set documentation to understand the problems or topics covered.  
    3. Launch Jupyter Notebook for interactive tasks or run Python scripts directly, adjusting paths as needed:
       ```
       jupyter notebook <path-to-notebook>.ipynb
       ```
       or
       ```
       python <path-to-script>.py
       ```
